### BookLog

- 독서록을 작성할 수 있는 애플리케이션입니다.
- 책에 대한 정보는 네이버 검색 API를 사용했습니다.
- URL
    - API: [https://api.booklog.dev/](https://api.booklog.dev/)
    - Page: [https://booklog.dev/](https://booklog.dev/)

### 사용 기술 및 환경 - Backend

- NestJS, TypeORM
- jest, super-test, ts-mockito
- Azure VM, Azure Database for MySQL
- Nginx, Jenkins
- Sonarcloud

### 사용 기술 및 환경 - Frontend

- Nuxt3
- Cloudflare Pages
- Sonarcloud
